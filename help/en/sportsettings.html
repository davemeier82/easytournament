Number of periods

The number of periods that a game in you tournament has (e.g. 2 for the two half times in soccer or 3 for the three thirds in icehockey).

Number of overtimes

Is it possible that there is an overtime? Is this the case, then you can set the number of maximal possible overtimes (e.g. 2 for the two overtime half times in soccer, 15 min. each, or 1 for the overtime in icehockey).

Points

In this fields you can set the amount of points a team gets with a win, draw or defeat.
