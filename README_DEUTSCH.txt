EasyTournament BETA-VERSION
============================

Vielen Dank dafür, dass sie sich für EasyTournament entschieden haben.


Es handelt sich hier um eine frühe Testversion von EasyTournament.
Wir raten ihnen dringend davon ab, diese Version als alleiniges Berechnungsprogramm 
für ein Turnier zu verwenden, das es zurzeit noch diverse Fehler enthalten kann.

Falls sie einen Fehler finden, bitten wir sie uns diesen mitzuteilen, damit wir ihnen
baldmöglichst ein stabiles Produkt liefern können.

Fehler und Verbesserungswünsche können sie uns wie folgt mitteilen:
- E-Mail an support@easy-tournament.com
- Absenden des Error-Reports falls ein Fehler auftritt
- Schreiben einer Nachricht im Forum auf easy-tournament.com/forum
