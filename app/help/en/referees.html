On this screen you are able to add referees to the tournament. You can do this by clicking on the "New Referee" button or by selecting the corresponding entry in the context menu (right mouse button).

With a double-click on a referee or with the context menu you can edit a referee.

The button "Import" allows you to import referees from another tournament.

Additionally you can save the referees in an XML file to use them in another application. To do this you have to click on the "Export" button.

New referee / Edit referee

Logo

To select a logo for the referee you have to click on the squared button on the upper right side and choose the desired logo.

If you like to delete the logo, then click with the right mouse button on the logo and select the entry "Reset logo" in the context menu.

Assistants

In the tab "Assistants" you are able to add some assistants to the referee. To do this you proceed exactly the same as with the referees. Further details about the assistants you can get in the chapter Assistants.
